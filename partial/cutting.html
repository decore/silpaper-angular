Резка рулонной продукции

1. Резка различных видов бумаг:

  • офсетная - применяется для печати, в том числе художественных изданий, книг, периодических изданий; изготавливается офсетным способом (перенос краски на материал осуществляется не напрямую);
  • крафт-бумага - особо прочная оберточная бумага, используемая для упаковки посылок, книг, некрупных предметов интерьера;
  • фильтровальная - применяется для фильтрации воды, масел и других веществ, содержащих различные примеси;
  • бумага для высокохудожественных изделий (ВХИ) - используется для печати журналов, книг, топографических карт шестью или более красками;
  • силиконизированная - бумага с силиконовым покрытием, служащая для защиты поверхности материала от самопроизвольного приклеивания;
  • крепированная (креп-бумага) - гофрированная бумага, применяемая для упаковки цветов и подарков;
  • самоклеящаяся бумага - предназначена для различных этикеток, ценников, плакатов, декоративных работ.

2. Резка различных видов пленок:

  • BOPP (БОПП), или биаксиально-ориентированная полипропиленовая пленка, - один из наиболее востребованных видов упаковочной пленки для гибкой упаковки, отличается высокой термостойкостью, прочностью, гибкостью;
  • PET (ПЭТ) - пленка, ламинированная полиэтиленом, применяется для изоляции электропроводящих материалов, производства упаковки, служит основой для упаковочных материалов;
  • CPP (каст-пленка) - один из основных упаковочных материалов для хлебобулочных изделий, защищая их от образования влаги. бактерицидного заражения;
  • PVC (ПВХ) - термоусадочная пленка, производимая из полихлорвинила, применяемая для упаковки маловесных и небольших по размеру изделий.

3. Резка комбинированных материалов, в состав которых входят материалы различного типа (бумага, картон, фольга, искусственные материалы).

4. Резка нетканого полотна, изготовленного из одного или нескольких видов текстильных материалов со скрепленной текстурой элементов.
